# GRAB Senior Helper

## Overview
The GRAB Senior Helper app is designed to provide assistance to elderly individuals by facilitating their daily activities and ensuring their safety. The app connects seniors with caregivers who can help them with various tasks, including transportation, grocery shopping, and companionship.

## Features
- **User-Friendly Interface:** Simple navigation for ease of use by seniors.
- **Caregiver Matching:** Connects users with available caregivers based on their needs.
- **Task Scheduling:** Allows seniors to schedule tasks easily with their caregivers.
- **Emergency Alerts:** Sends alerts to caregivers in case of an emergency.
- **Location Tracking:** Enables caregivers to track the location of seniors during outings.

## Setup Instructions
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/parthanadathur1307-cmyk/grab-senior-helper.git
   ```
2. **Install Dependencies:**
   Navigate into your project directory and install the required packages:
   ```bash
   cd grab-senior-helper
   npm install
   ```
3. **Configuration:**
   Update the configuration settings as needed in the `config.js` file.

4. **Run the Application:**
   ```bash
   npm start
   ```

## How to Use the App
1. **Sign Up:** Create an account as a senior or a caregiver.
2. **Profile Setup:** Fill out your profile with necessary details.
3. **Searching for Caregivers:** If you are a senior, use the search functionality to find available caregivers.
4. **Scheduling Tasks:** Schedule and confirm tasks with your selected caregiver.
5. **Use Emergency Features:** Utilize the emergency alert features whenever necessary.

For more information, please refer to the user manual available within the app.
